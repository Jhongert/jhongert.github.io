Eat-Da-Burger

A full stack burger-logger app. On the server side this app uses Node and MySQL to query and route data and Handlebars to generate HTML. this app also follows the MVC design pattern and use Sequelize to interact with the database. On the client side uses Bootstrap for layout and style, and JQuery to validate user input.

Developed with Node.js, Express, MySQL, Sequelize, JQuery and Bootstrap.

Friend Finder

Friend Finder is an app that helps you find your most compatible friend. This app will take the result from a survey, then compare the answers with those from other users. The app will then display the name and picture of the user with the best overall match.

Developed using Express and Node.js.
